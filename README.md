📖 使用說明
本工具彙整 2026 TSA 台灣外科醫學會年會暨外科聯合學術國際研討會的所有演講議程，提供四種檢視方式。

🔍 隨選查詢
依日期、時段、科別、會場、類型、關鍵字篩選。關鍵字會同時比對題目、講者、服務單位、主持人。多個條件可同時使用。

⏰ 當下節目表
自動顯示當下(或指定模擬時間)正在進行的節目，以及接下來的三個時段預告。適合會議現場參考。

📊 時間軸檢視
以表格方式橫向呈現同一時段各會場的並行議程，方便比較選擇。

📦 資料範圍
各科 Special Lecture、Symposium、Oral Presentation (共 698 場)
已略過僅標示「Discussion」、「Opening/Closing Remarks」、「會員大會」、頒獎等無個別講題之場次
資料僅供參考，實際議程以現場公告為準
💾 離線使用
本檔案為完全離線運作 (所有資料、CSS、JS 寫死在單一 HTML 檔中)，無需網路連線。可儲存至手機或電腦直接開啟使用。
